Guide d'Utilisation du Bookmarklet

Introduction

Ce document explique comment ajouter un bookmarklet à vos favoris et comment l'utiliser après avoir hébergé le code nécessaire.

Ajouter le Bookmarklet aux Favoris

1. Créer le Bookmarklet

Copiez le code du bookmarklet. Il devrait ressembler à quelque chose comme :

javascript:(function() {
  // Code du bookmarklet ici
})();

Assurez-vous que le code est complet et correctement formé.

2. Ajouter aux Favoris

Ouvrez votre navigateur web.

  • Pour Chrome : Cliquez sur l'étoile dans la barre d'adresse et sélectionnez "Ajouter aux favoris".
  • Pour Firefox : Cliquez sur l'étoile dans la barre d'adresse et sélectionnez "Ajouter aux favoris".
  • Pour Edge : Cliquez sur l'étoile dans la barre d'adresse et sélectionnez "Ajouter aux favoris".

Dans le champ "URL" ou "Adresse", collez le code du bookmarklet.

Donnez un nom au bookmarklet pour le reconnaître facilement.

Enregistrez le favori.

Héberger la Seconde Partie du Code

1. Hébergement

Hébergez la seconde partie du code (le script ou les ressources nécessaires) sur un serveur web. Vous pouvez utiliser des services d'hébergement gratuits ou payants, selon vos besoins.

Assurez-vous que le code est accessible publiquement via une URL.

2. Mettre à Jour le Bookmarklet

Modifiez le code du bookmarklet pour inclure l'URL où la seconde partie du code est hébergée. Par exemple :

javascript:(function() {
  var script = document.createElement('script');
  script.src = 'https://raw.githubusercontent.com/tucommenceapousser/exturls/main/exturl.js';
  document.head.appendChild(script);
})();

Remplacez https://raw.githubusercontent.com/tucommenceapousser/exturls/main/exturl.js par l'URL réelle où le code est hébergé.

http://counottes.fr

3. Tester

Accédez à une page web de votre choix.

Cliquez sur le bookmarklet dans vos favoris.

Vérifiez que le code s'exécute correctement et que la fonctionnalité attendue est activée.

Conclusion

Vous avez maintenant ajouté le bookmarklet à vos favoris et configuré l'accès à la seconde partie du code. Assurez-vous que tout fonctionne correctement en testant le bookmarklet sur différentes pages web.

Pour toute question ou problème, n'hésitez pas à consulter la documentation du bookmarklet ou à contacter le support technique.
